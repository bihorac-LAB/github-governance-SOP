# Automation & GitHub Actions ("Robots")

This section documents the **GitHub Actions workflows** used to enforce governance and safety:

- PHI/PII scanner
- Secrets scanner
- Repository structure checker
- Publication readiness checker
- Labeling / classification

For conceptual details, see Section 9 of the [master manual](../sop/master_manual.md#9-automation-and-robots-github-actions).

Implementation examples (YAML) can live in this section as you build them out.
