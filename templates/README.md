# Repository Templates

Copy-paste templates for your repositories.

## Quick Download

```bash
# README template
curl -o README.md https://raw.githubusercontent.com/bihorac-LAB/github-governance-SOP/main/templates/README.template.md

# Required files
curl -o DATA_CLASSIFICATION.md https://raw.githubusercontent.com/bihorac-LAB/github-governance-SOP/main/templates/data-classification-template.md
curl -o CONTRIBUTING.md https://raw.githubusercontent.com/bihorac-LAB/github-governance-SOP/main/templates/CONTRIBUTING.md

# GitHub templates
curl -o .github/pull_request_template.md https://raw.githubusercontent.com/bihorac-LAB/github-governance-SOP/main/templates/pull-request-template.md
```

## Available Templates

**README Template:**
- `README.template.md` - One template for all projects (customize as needed)

**Required Files:**
- `data-classification-template.md` - Data classification (REQUIRED)
- `CONTRIBUTING.md` - Contribution guidelines
- `access-request-template.md` - Access request form

**GitHub Templates:**
- `pull-request-template.md` - Standard PR template
- `issues/` - Bug report, feature request templates
- `repository-files/` - SECURITY.md, LICENSE files

**Other:**
- `licenses/` - Different license templates
- `compliance/` - Compliance documentation templates

Browse all: https://github.com/bihorac-LAB/github-governance-SOP/tree/main/templates
